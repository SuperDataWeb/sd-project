速达3000  BAS 详细功能介绍:

采购管理


核心功能:

采购订单管理：直接开单及根据销售订单生成采购订单、采购订单导出、采购订单直接发送邮件、超期订单提醒、中止订单
采购收货管理：直接开单及根据采购订单生成采购收货单、整单折扣及单品折扣、采购收货单直接发邮件、应付账款超期提醒、库存上下限提醒、采购最高进货价提醒、付款、退货流程查看
采购付款管理：多种付款类型、多种付款方式、整单付款及部份付款、总部对分支机构供应商付款、采购付款直接发邮件
采购退货管理：退还现款、冲抵往来、退货数量、金额不符提醒、采购退货直接发送邮件、查看付款流程

核心报表：（部分）
采购订单汇总表、采购订单明细表、供应商供货汇总表、供应商供货明细表、应付账款汇总表
商品采购汇总表、商品采购明细表、业务员采购汇总表、业务员采购明细表、应付账款明细表


销售管理


核心功能:

销售订单管理：直接开单及Excel导入生成销售开单、销售订单直接发送邮件、销售订单转入采购订单、超期订单提醒、中止订单
销售开单管理：直接开单及根据销售订单生成销售开单、现款销售管理、整单折扣及单品折扣、销售开单直接发邮件、应收账款超期提醒、库存上下限提醒、销售最低价提醒、销售员业绩管理、销售员敏感数据保护、客户信誉额度控制、超期应收款客户控制、零库存开单控制、收款、退货流程查看
销售收款管理：多种收款类型、多种收款方式、整单收款及部份收款、总部对分支机构客户收款、销售收款直接发邮件
销售退货管理：退还现款、冲抵往来、退货数量、金额不符提醒、销售退货直接发送邮件、查看收款流程

核心报表：（部分）
客户销售汇总表、客户销售明细表、商品销售汇总表、商品销售明细表、应收账款汇总表、仓库销售汇总表
业务员销售汇总表、业务员销售明细表、销售毛利汇总表、销售毛利明细表、地区销售汇总表


POS零售管理（后台）



核心功能:

POS销售单管理、POS出入款单管理、POS日结、POS日结单、POS机品设置及管理
会员积分规则设置、会员积分调整、会员积分返现、兑换奖品

核心报表：（部分）
POS收银员收银汇总表、POS收银员收银明细表、POS销售汇总表、POS收银员销售明细表
POS货品销售明细表、POS流水账、POS收银员交款汇总表、POS收银员交款明细表

POS零售管理（前台）




核心功能:

在线或离线两种工作方式、支持常规外设、收款、挂单、锁机、换班、折扣、复折、删除、即清、支持多种收款方式

仓库管理



核心功能：

领料管理、退料管理、产品进仓管理
货品调拨管理：分支货品调拨、总部分支调拨、仓库间调拨、同价调拨
仓库盘点管理：仓库实物盘点、仓库盈亏数量处理、自动结转盈亏金额、支持盘点设备
总仓核算管理、仓库资料管理、分仓库存管理、库存变动类型管理

核心报表：（部分）
帐面库存汇总表、帐面库存明细表、帐面库存数量汇总表、帐面库存数量明细表
实际库存数量汇总表，实际库存数量明细表,实际库存数量分类表，分仓库存报表


分销管理（加盟商）   *需要与分销管理（V300+.net）配合使用。



核心功能:

分销机构加盟：分销机构加盟后与总部和其他分销机构组成分销体系。
分销数据下载：分销机构下载总部授权货品信息、分销订单处理结果等。
分销数据上传：分销机构上传分销订单、分销销售数据、分销库存数据等信息。
分销采购订单：分为两种流程，既可以由分销机构发起，也可以由企业总部发起。
分销销售订单：分为两种流程，既可以由分销机构发起，也可以由企业总部发起
分销货品授权：分销机构经授权可以查询其他机构的分销货品库存。

核心报表：
分销机构库存明细表、分销货品多机构库存、分销机构销售汇总表、分销机构销售明细表

网店管理   *本功能暂未推出。




核心功能:

网店注册、下载商品、数据同步设置、网店订单、网店发货单、网店收款单、网店退换货、网店付款单以及财务处理和报表统计


现金银行



核心功能:

费用开支：有效管控企业各种费用支出，监控收款、付款过程。
其他收入：管理除正常经营业务收入外的其他收入，提供“往来结算、现款结算”两种类型。
银行存取款：出纳存取现金、票据资金转存银行、分支资金转入总部、总部资金下拨分支等业务下理。

核心报表：
日记账汇总表、日记账明细表、收入类别报表、支出类别报表


财务系统    *可与进销存结合使用也可分开单独使用。



核心功能:

凭证录入：提供仿真凭证录入方式及智能凭证查找，作账更轻松。
凭证审核：提供单张、批量凭证审核及反审核方式，高效凭证登账。
期末调汇：核算外币科目、调整汇兑差异、现金流量分配等业务。
结转损益：损益科目余额结转、支持多次结转处理。
期末结账：自动结转销售成本、自动结转生产成本、自动结转代销成本、自动结转发出成本、结转委找加工成本、结转组装拆卸成本。

核心报表：（部分）
资产负债表、利润表、现金流量表、试算平衡表、科目汇总表
日记账、明细分类账、多栏账、总分类账

其他重要模块

权限设置 ：软件提供严格的权限设置，加强内部控制和监督管理。
账套选项：为企业提供更多企业管控需求设置。
经营分析：提供采购、销售、库存、往来等分析，为经营决策者提供决策数据。
打印样式设计：为企业提供专属的单据样式，企业徽标、凭证套打等，正规，专业。
系统维护：账套备份、恢复、初始数据导入、账套收缩、隐藏账套等。
